gitflow:hotfix-finish

Full name:

com.amashchenko.maven.plugin:gitflow-maven-plugin:1.20.0:hotfix-finish

Description:

The git flow hotfix finish mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
