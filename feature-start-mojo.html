gitflow:feature-start

Full name:

com.amashchenko.maven.plugin:gitflow-maven-plugin:1.17.0:feature-start

Description:

The git flow feature start mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
