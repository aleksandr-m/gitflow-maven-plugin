gitflow:feature-finish

Full name:

com.amashchenko.maven.plugin:gitflow-maven-plugin:1.19.0:feature-finish

Description:

The git flow feature finish mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
